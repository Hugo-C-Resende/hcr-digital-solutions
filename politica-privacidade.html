Política de Privacidade

Última atualização: 2026

1. Introdução

A HCR Digital Solutions respeita a tua privacidade e está comprometida em proteger os teus dados pessoais. Esta política explica que dados podem ser recolhidos, como são utilizados e quais os teus direitos.

2. Dados recolhidos

Este website pode recolher dados básicos como:

  • Nome e email quando nos contactas por email;
  • Informação técnica básica (como tipo de browser e sistema operativo) através de ferramentas de análise;
  • Dados incluídos voluntariamente em mensagens enviadas para nós.

3. Finalidade do tratamento

Os dados são utilizados para:

  • Responder a pedidos de contacto;
  • Melhorar o conteúdo e a experiência do website;
  • Cumprir obrigações legais aplicáveis.

4. Partilha de dados

Não vendemos nem cedemos os teus dados a terceiros, exceto quando exigido por lei ou quando necessário para prestação de serviços (por exemplo, serviços de alojamento).

5. Conservação dos dados

Os dados são conservados apenas pelo período necessário para cumprir as finalidades para que foram recolhidos ou enquanto existir obrigação legal.

6. Direitos do titular dos dados

Tens direito a:

  • Aceder aos teus dados pessoais;
  • Solicitar a retificação ou atualização;
  • Solicitar o apagamento, quando aplicável;
  • Opor-te a determinado tratamento;
  • Apresentar reclamação junto da autoridade de controlo competente.

7. Contacto

Para questões relacionadas com privacidade, podes contactar:
