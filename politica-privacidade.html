Política de Privacidade

A HCR Digital Solutions respeita a privacidade dos utilizadores e garante que os dados pessoais são tratados de forma segura e responsável.

1. Recolha de Dados

Recolhemos apenas os dados fornecidos voluntariamente através do formulário de contacto.

2. Finalidade

Os dados são utilizados exclusivamente para resposta a pedidos de contacto.

3. Direitos do Utilizador

O utilizador pode solicitar a eliminação ou alteração dos seus dados a qualquer momento.

4. Segurança

Implementamos medidas técnicas para proteger os dados contra acesso não autorizado.
